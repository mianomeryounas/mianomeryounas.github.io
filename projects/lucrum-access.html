Lucrum Access Control

A ZKTeco biometric integration with Frappe ERP.

Project Overview

Lucrum Access Control is a specialized application that integrates ZKTeco biometric systems with Frappe ERP, providing comprehensive access control and attendance management solutions.

Tech Stack

Flutter Dart Getx (State Management & Routing & DI) Clean Architecture SOLID Principles RESTful APIs WebSockets (Socket.io) Firebase (Crashlytics, Analytics) ZKTeco SDK Integration Frappe Integration

Key Features

  • Biometric Integration
    • Fingerprint recognition
    • Face recognition
    • Card-based access
    • Multi-device support
  • Access Control
    • Door access management
    • Time-based access
    • Area restrictions
    • Access logs
  • Attendance Management
    • Automated attendance tracking
    • Leave management
    • Overtime calculation
    • Report generation
  • ERP Integration
    • Real-time data sync
    • Employee management
    • Department integration
    • Report automation

Project Gallery

Development Process

The project was built using Clean Architecture principles with GetX for state management. Special focus was placed on biometric integration, real-time synchronization, and security.

Impact & Results

  • Improved access control efficiency
  • Reduced manual attendance errors
  • Enhanced security measures
  • Streamlined HR operations
